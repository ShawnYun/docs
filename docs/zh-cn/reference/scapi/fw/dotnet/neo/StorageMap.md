# StorageMap 类

表示给定存储上下文中指定前缀的 key-value 存储区。

命名空间：[Neo.SmartContract.Framework.Services.Neo](../neo.md)

程序集：Neo.SmartContract.Framework

## 语法

```c#
public class StorageMap
```

## 属性

| 名称    | 说明       |
| ------- | ---------- |
| Context | 存储上下文 |
| Prefix  | 前缀       |

## 构造方法

通过 [Helper](Helper.md) 的 CreateMap 方法来构建 StorageMap 对象。
